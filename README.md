# CRATE — симулятор кейсов CS2 с реальными ценами Market.CSGO

Статический сайт (HTML/CSS/JS, без бэкенда). В кейсах — реальные предметы CS2
с картинками и живыми ценами в рублях с [market.csgo.com](https://market.csgo.com).
Клик по предмету открывает Market.CSGO с фильтром по этому скину (все износы:
прямо с завода, немного поношенное, после полевых испытаний, поношенное,
закалённое в боях, а также StatTrak™ / Souvenir).

## Откуда берутся цены

Прямой запрос `market.csgo.com/api/v2/prices/RUB.json` из браузера блокируется
CORS. Поэтому цены парсятся скриптом и кладутся в статический файл:

- **`scripts/generate_data.py`** — парсит цены (market.csgo.com) и базу предметов
  [ByMykel/CSGO-API](https://github.com/ByMykel/CSGO-API) (картинки, редкость,
  износы), собирает `data/items.json`. Только стандартная библиотека Python.
- **`data/items.json`** — сгенерированные данные: 5 кейсов с реальными предметами,
  ценами, картинками и ссылками на маркет. Сайт грузит его как обычный файл.
- **`.github/workflows/update-prices.yml`** — GitHub Action раз в сутки запускает
  генератор и коммитит свежий `data/items.json`. Бесплатно.

Цены обновляются раз в сутки (не real-time). Время последнего обновления
показывается на страницах кейсов («Цены обновлены: …»).

## Цена кейса

`price = EV × 1.18`, где `EV` — ожидаемая стоимость выпадения при взвешенном
ролле по редкости (Mil-Spec 79.92%, Restricted 15.98%, Classified 3.2%,
Covert 0.64%, ножи/перчатки 0.26%). Перевес казино ~15%: большинство спинов
проигрышные (дешёвые предметы дешевле кейса), редкие ножи дают крупный выигрыш.

## Локальный запуск

Открыть `index.html` через любой статический сервер (нужен для fetch `data/`):

```bash
python3 -m http.server 8000
# http://localhost:8000
```

Регенерировать данные вручную:

```bash
python3 scripts/generate_data.py
```
